Sommersloops

              Set up a semi-automated production for hand gathered items using sloops for 
              * gathered slugs => container => power shards => container
              * gathered alien remains => container => alien protein => alien dna => container
              * gathered foliage => container => biomass => solid biofuel => container

              Use sloops to double the production output at each step!
              you only need one, which you can pop into each machine as needed

              You must have the sloop in the machine before starting production!!! 
              I construct/deconstruct input belts to prevent accidentally starting production without sloops
